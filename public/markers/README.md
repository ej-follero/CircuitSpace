# AR Marker Patterns

This directory contains AR marker patterns for use with AR.js marker detection.

## Available Markers

### Hiro Marker (Default)
- **Pattern**: Hiro
- **Download**: [Hiro Marker Image](https://jeromeetienne.github.io/AR.js/data/images/HIRO.jpg)
- **Usage**: Most commonly used marker pattern
- **Print Size**: Recommended 8cm x 8cm (3.15" x 3.15")

### Kanji Marker
- **Pattern**: Kanji
- **Download**: [Kanji Marker Image](https://jeromeetienne.github.io/AR.js/data/images/KANJI.jpg)
- **Usage**: Alternative marker pattern
- **Print Size**: Recommended 8cm x 8cm (3.15" x 3.15")

## How to Use

1. Download or print one of the marker patterns above
2. Ensure good lighting when using the marker
3. Keep the marker flat and visible to the camera
4. Start AR mode in the application
5. Point your camera at the marker to see 3D IoT devices

## Tips

- Use a well-lit environment
- Keep the marker flat (not bent or wrinkled)
- Maintain a distance of 20-50cm from the marker
- Ensure the marker is fully visible in the camera frame
- Avoid reflections or shadows on the marker
